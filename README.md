



# ServiceNow Ticketing Lab

## My Objective

To simulates Level 1 Help Desk ticketing workflows using ServiceNow ITSM. The goal is to demonstrate incident creation, prioritization, documentation, escalation, and resolution for common end-user technical issues.

## The Tools Used

- ServiceNow Personal Developer Instance
- Incident Management
- Assignment Groups
- Work Notes
- Resolution Notes

## Skills Demonstrated

- Creating and managing incident tickets
- Documenting troubleshooting steps
- Prioritizing requests based on impact and urgency
- Assigning tickets to support groups
- Escalating issues when needed
- Resolving and closing incidents with clear notes

## Lab Scenarios

This lab will include simulated tickets for:

1. Password reset / account lockout
2. Wi-Fi connectivity issue -[INC000001 - Wi-Fi Connectivity Issue](./documentation/INC000001%20-%20Wi-Fi%20Connectivity%20Issue)  90%  Documentation Complete / ServiceNow Pending on Instance waiting list. |
3. SOC -Multiple Failed log in Attempts.
4. SOC- Repeated Unauthorized Connection Attempts from External IP
5. SOC- Unauthorized Privilege Escalation
